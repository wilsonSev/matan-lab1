## Results

В ходе работы была построена последовательность функций:

`f_n(x) = floor(2^n * f(x)) / 2^n`

Исследовано её поведение на графиках и выполнено вычисление интегралов.

Численные результаты для больших `n` (`100`, `1000`) практически совпали с аналитическими значениями как для интеграла Лебега, так и для интеграла Лебега–Стилтьеса (при выбранной функции `F(x)` и корректном отрезке интегрирования).

Наблюдается ожидаемая сходимость: при росте `n` погрешность уменьшается и стремится к нулю, что подтверждает теоретические выводы и корректность применённого численного метода.
